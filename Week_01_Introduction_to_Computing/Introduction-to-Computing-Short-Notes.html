06 — Memory & Storage
Primary memory is inside the computer. It works directly with the CPU.
🧠
RAM
Working memory. VOLATILE — wiped when power off.
📖
ROM
Startup instructions. PERMANENT — never erased.
⚡
Cache
Fastest memory. Sits between CPU & RAM.
Secondary storage keeps data permanently even when the computer is off.
🧲

08 — Software
Click each section to expand it
🖥️
System Software
Manages the computer itself
▼
Runs in the background. Without it, hardware does nothing. Includes the Operating System (OS) and utility programs.
⊞ Windows 🍎 macOS 🐧 Linux 🛡️ Antivirus 🔧 Disk Tools
📦
Application Software
Programs users actually run
▼
Helps users do real tasks — write, calculate, browse, design, play.
📝 MS Word 📊 Excel 🌐 Chrome 🎮 Games 🎨 Photoshop 📧 Email
🖱️
OS Interfaces: CLI vs GUI
Two ways to talk to a computer
▼
⌨️ CLI — Command Line
You type text commands. Fast but requires memorising commands.
🖱️ GUI — Graphical
Click icons & menus. Easier to use. Used by most people today.
09 — Computer Ergonomics
👁️
Monitor
Eye level, about arm's length away. Top of screen below eye level.
💡 Prevents neck strain
🪑
Chair & Posture
Back supported, knees at 90°+, feet flat on the floor.
💡 Prevents back pain
⌨️
Keyboard
Forearms horizontal, wrists straight when typing.
💡 Prevents wrist injury
💡
Lighting
Light source to the side — never directly behind the screen.
💡 Prevents eye strain
